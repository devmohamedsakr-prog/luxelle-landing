## 🚀 Quick Start

If you're looking to jump right in:

```bash
# 1. Clone the repository
git clone https://github.com/Mostafa-SAID7/luxelle-landing-page.git

# 2. Install Dependencies
cd luxelle-landing-page
npm install

# 3. Start the application
npm start
```
*Navigate to http://localhost:4200/ in your browser.*
